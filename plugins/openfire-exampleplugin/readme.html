Example Plugin

This is an example plugin, written against Openfire 4.2 using the Maven build system

Structure

Top Level
The top level directory may contain a read.html (this file), a changelog.html, a logo_large.git (and logo_small.gif), and finally the plugin.xml - the latter being required.
src/main/java
This contains sources.
src/main/i18n
This contains internationalisation files. This example ships with exampleplugin_i18n.properties - the default language (expected to be English), and exampleplugin_i18N_fr.properties for a sample French translation. Specific language translation files may be incomplete, and may be specific to a full locale (such as fr_CA for Quebecois).
src/main/web
This directory contains JSP and assets for the admin console. The plugin.xml in the top level includes how to link these pages into the Admin Console itself.
src/main/database
Contains plugin database schema and any upgrade scripts
